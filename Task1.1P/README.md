This project implements a simple smart lighting system using Arduino Nano 33 IoT.
The system uses a push button as input and two LEDs to represent the porch light and the hallway light.

When the button is pressed, the porch light turns on for 30 seconds and the hallway light turns on for 60 seconds.

The program is organised into logical parts to make it easier to understand.
One part of the code reads the button input, another part controls the LEDs, and another part handles the timing.

This modular approach makes the program easier to modify and test.
